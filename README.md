# lxd-zabbix
Install Zabbix in LXC behind Nginx Reverse Proxy

# Prerequisites
- Ubuntu 20.04 (Recommended)
- LXD init
- FQDN

# Usage
```
git clone https://github.com/antyung88/lxd-zabbix.git && cd lxd-zabbix
```
Edit variables
```
sudo nano zabbix.sh
```
Deploy
```
chmod +x zabbix.sh
./zabbix.sh
```
